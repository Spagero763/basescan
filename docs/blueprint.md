# **App Name**: BaseScan

## Core Features:

- Metrics Dashboard: Display aggregated DeFi metrics, including TVL, 24h volume, and number of users, for the Base ecosystem.
- Token Prices: Display price, volume, and market cap for tokens tracked in the aggregator.
- Protocol Listing: List all DeFi protocols monitored by the app.
- APY Calculation: Display a calculated APY (Annual Percentage Yield) for each protocol, estimated based on the TVL and trading volume. The application shall have a field to take in an API key.
- AI Insights: AI-powered summarization tool that analyzes and condenses DeFi metrics to generate key insights and trends for the Base ecosystem, utilizing a tool that interprets the aggregated data.
- Filtering: Filter protocols by category (e.g., DEX, lending, etc.) to refine the displayed data.
- Etherscan Linking: The app allows the user to enter a contract address to look for more info. The app renders external links to Etherscan and other useful websites.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) – A vibrant blue (#3399FF) to evoke trust and stability.
- Background color: HSL(210, 20%, 20%) – A dark, muted blue (#33394D) to create a professional, high-tech feel.
- Accent color: HSL(240, 80%, 60%) – A deep purple (#4D4DFF) to highlight key data and interactive elements, creating visual interest.
- Body font: 'Inter', sans-serif, for a clean, modern, and readable interface.
- Headline font: 'Space Grotesk', sans-serif, offering a computerized feel for headlines.
- Use minimalistic line icons to represent different DeFi protocols and metrics.
- Data displayed in a clean, card-based layout to emphasize structure.
